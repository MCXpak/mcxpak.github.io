👩🏽‍🚀 Projects

Typespace - Typing Practice Game

Link

Want to get good at typing but bored of popular typing trainers? Typespace is a typing trainer where you shoot incoming asteroids with a spaceship. This project involved string matching, animation, level design and general web design.

A Journey With Dijktra - Node Graph Maker and Shortest-Path Solver

Link

Producing node graphs by hand can be a pain, by and or computer. This quick and simple solution allows the user to produce quick graphs, returning the output of the graph and shortest path as a string object.

BrainBase - Educational Humanities Games website for Kids

Link

Making learning humanities fun, BrainBase brings joy to learning. Created as part of final bootcamp project. Currently 2 games are hosted on the website. Created a point-click adventure game from scratch using react drag & drop framework.

KnowMyGrade - A University Grade Calculator

Link

Most university grade calculators either come with too much or too little. KnowMyGrade finds the sweet spot that allows a student to enter all their grade and credit information without clicking through too many windows. Clean and intuitive UI allows for a hassle free input experince.
